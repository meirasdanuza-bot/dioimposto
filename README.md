# Planilha de Imposto de Renda em Excel

Este projeto consiste no desenvolvimento de uma planilha em Microsoft Excel para organização e centralização de informações necessárias à declaração do Imposto de Renda.

## Objetivo
Auxiliar o usuário no controle e registro de dados relevantes para a declaração, reunindo as informações em um único local de forma estruturada, validada e de fácil navegação.

## Funcionalidades
- Agregador de dados financeiros
- Validação automática de campos
- Menus de navegação para facilitar o uso
- Links rápidos para acesso a informações
- Interface simples, prática e amigável

## Ferramentas utilizadas
- Microsoft Excel

## Observações
Projeto desenvolvido como atividade prática em curso de Excel, aplicando conceitos de organização de dados, validações, usabilidade e automação básica dentro do ambiente Excel.
